# Gizlilik Politikası

Son güncelleme: 21 Haziran 2026

Weather Decision Agent bir portföy ve demo uygulamasıdır. Hesap oluşturmaz;
ad, e-posta veya telefon numarası istemez.

Uygulama, öneri üretmek için seçtiğiniz konumu veya şehir adını, hava ve
aktivite tercihlerinizi ve verdiğiniz öneri geri bildirimini işler. Herkese açık
demoda öneri geçmişi yalnızca geçici oturum dosyalarında tutulur ve kalıcı
kullanıcı profili oluşturulmaz.

Hava verileri Open-Meteo'dan, mekan sonuçları Google Places'tan ve harita
Google Maps JavaScript API'den alınabilir. LLM özelliği etkinleştirildiğinde,
öneriyi açıklamak için gerekli sınırlı içerik yapılandırılan LLM sağlayıcısına
gönderilebilir. Bu hizmetlerin kendi gizlilik koşulları geçerlidir.

Uygulamanın kaynak kodu ve veri işleme akışı GitHub deposunda incelenebilir:
https://github.com/dozmen23/weather-decision-agent-v2

Gizlilikle ilgili sorular için GitHub deposunda issue açabilirsiniz.
